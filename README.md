 Terminal Portfolio

An interactive terminal-style personal portfolio for Sukhman Shergill deployed on Netlify:

Live site https://teminalsukhi.netlify.app/

 Overview

This project is a browser-based portfolio designed to feel like a real terminal. Visitors can type commands or use quick command buttons to explore sections like:

- About
- Resume
- Projects
- Goals
- Resume download
- LinkedIn

The interface combines a terminal-inspired layout with a clean neon-green visual style and simple interactive JavaScript.

 Features

- Terminal-inspired portfolio UI
- Interactive command input
- Quick command chips in the header
- Typing-style output for most responses
- Resume download shortcut
- LinkedIn profile shortcut
- Clear terminal command
- Responsive layout for desktop and mobile
- Animated glowing background

 Available Commands

Type any of these commands into the terminal input:

- `help` — show all available commands
- `about` — display a short introduction
- `resume` — show education, experience, and skills
- `projects` — list featured projects
- `goals` — show career and learning goals
- `download` — open the resume link
- `linkedin` — open the LinkedIn profile
- `clear` — clear the terminal output
